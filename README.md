## Installation & Setup
To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/san98215/san98215.github.io.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
